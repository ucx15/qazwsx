# QAZWSX

3D rasterizer made in C++.

**⚠️Currently in Dev**

## Dependencies

* [stb_image_write.h](https://github.com/nothings/stb/blob/master/stb_image_write.h)
* [SDL3](https://www.libsdl.org/)
* [GLM](https://github.com/g-truc/glm)

## Quick Start

Setup SDL3 and GLM in [Libs](Libs/) directory.

Then-
```
$ ./build.ps1
$ ./qazwsx
```
